Политика конфиденциальности — Study Helper

Последнее обновление: 01 июля 2026 г.

1. Общие положения

Study Helper — это браузерное расширение, предназначенное для работы с учебными материалами, распознавания текста и получения подсказок с использованием сервисов искусственного интеллекта.

Расширение работает преимущественно локально на устройстве пользователя. Разработчик не собирает, не получает и не хранит пользовательские данные на собственных серверах.

2. Какие данные обрабатываются

Расширение может обрабатывать следующие данные:

текст вопросов и учебных материалов;
изображения, выбранные пользователем для распознавания текста (OCR);
загруженные пользователем документы;
настройки расширения;
локальную базу ответов и результатов обработки;
API-ключи, предоставленные пользователем для доступа к сторонним AI-сервисам.
3. Где хранятся данные

Следующие данные хранятся исключительно на устройстве пользователя:

настройки расширения;
API-ключи;
локальная база ответов;
загруженные документы;
результаты OCR;
история работы расширения.

Для хранения используются механизмы браузера, включая:

chrome.storage.local;
IndexedDB.

Разработчик не имеет доступа к этим данным.

4. Передача данных сторонним сервисам

По запросу пользователя расширение может отправлять данные напрямую выбранному провайдеру искусственного интеллекта для генерации ответов или обработки информации.

Поддерживаемые провайдеры могут включать:

Anthropic (Claude);
Groq;
xAI (Grok);
GigaChat;
иные провайдеры, добавленные в будущих версиях.

Передаваемые данные могут включать:

текст вопроса;
распознанный текст;
фрагменты загруженных учебных материалов;
изображения, если это необходимо для выполнения запроса.

Передача осуществляется напрямую между браузером пользователя и выбранным AI-провайдером с использованием API-ключа пользователя.

Разработчик не участвует в обработке, передаче или хранении таких данных.

5. Что расширение не делает

Study Helper:

не собирает персональные данные пользователей;
не отправляет данные разработчику;
не использует рекламные сети;
не осуществляет отслеживание действий пользователей;
не продаёт и не передаёт данные третьим лицам в маркетинговых целях;
не использует аналитику для профилирования пользователей.
6. Разрешения браузера

Расширение может использовать следующие разрешения:

storage

Для сохранения настроек пользователя и локальных данных.

unlimitedStorage

Для хранения больших объёмов учебных материалов, OCR-данных и локальной базы ответов.

host_permissions

Для доступа к поддерживаемым образовательным платформам и взаимодействия с выбранными пользователем AI-сервисами.

declarativeNetRequestWithHostAccess

Для управления сетевыми запросами, необходимыми для работы расширения.

Все разрешения используются исключительно для заявленных функций расширения.

7. Безопасность

Расширение использует защищённые HTTPS-соединения при взаимодействии со сторонними сервисами.

Несмотря на принимаемые меры безопасности, пользователь самостоятельно несёт ответственность за сохранность своих API-ключей и соблюдение требований выбранного AI-провайдера.

8. Права пользователя

Пользователь может в любое время:

удалить сохранённые данные через настройки расширения;
удалить API-ключи;
очистить локальную базу данных;
удалить расширение вместе со всеми связанными данными.
9. Изменения политики

Настоящая Политика конфиденциальности может обновляться. Актуальная версия всегда публикуется по адресу, указанному на странице расширения в Chrome Web Store и репозитории проекта.

10. Контакты

По вопросам, связанным с конфиденциальностью и обработкой данных, можно связаться с разработчиком:

Email: your-email@example.com

Используя Study Helper, пользователь подтверждает, что ознакомился с настоящей Политикой конфиденциальности и самостоятельно принимает решение о передаче данных выбранному провайдеру искусственного интеллекта в соответствии с его условиями использования и политикой конфиденциальности.
